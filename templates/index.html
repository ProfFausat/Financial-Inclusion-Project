How It Works

Model Type: Linear Support Vector Machine (SVM) with Stochastic Gradient Descent

Training Data: The model was trained on survey data from multiple African countries, analyzing demographic, economic, and technology usage patterns.

Input Features: 17 socioeconomic indicators are collected and transformed into 22 features through scaling, encoding, and missing value indicators.

Prediction Process: The model analyzes these features to calculate the probability that an unbanked individual will open a bank account, helping target financial inclusion efforts effectively.
